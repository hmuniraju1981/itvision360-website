# ITVision360 — Corporate Website

A premium dark-themed corporate website for **ITVision360 Inc.**, a technology solutions provider specializing in Industrial Automation, AI Strategy & Enablement, IoT & Cloud Solutions, and Enterprise IT Services.

## Tech Stack

- **Framework:** Next.js 14 (App Router, Static Export)
- **Styling:** Tailwind CSS (Dark Theme)
- **Icons:** Lucide React
- **Animations:** Framer Motion (ready for enhancement)
- **Font:** Inter (Google Fonts)
- **Deployment:** Netlify

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
